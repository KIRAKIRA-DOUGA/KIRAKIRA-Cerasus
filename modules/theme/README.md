# @KIRAKIRA-DOUGA/theme

> 原项目名称：**@nuxtjs/color-mode**
>
> 使用 Nuxt 轻松实现自动检测的🌑深色和🌕浅色模式。

KIRAKIRA 的主题配色模式模块基于官方 Nuxt Color Mode 模块修改而成。

除深浅色和系统自动外，增加了主题色改变的设置。

原版官网地址：[@nuxtjs/color-mode](https://color-mode.nuxtjs.org/)
