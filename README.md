## Nuxt 3

首先，Nuxt 读作 **/nʌkst/**（

查看 [Nuxt 3 文档](https://v3.nuxtjs.org) 以了解更多信息。

## 安装

确保安装依赖项：

```bash
# npm
npm install
```
**只准用 NPM（你怕吗）！！！**

## 开发服务器

从此链接启动开发服务器：http://localhost:3000

```bash
npm run dev
```

## 生产

为生产构建应用程序：

```bash
npm run build
```

本地预览生产版本：

```bash
npm run preview
```

有关详细信息，请查看[部署文档](https://v3.nuxtjs.org/guide/deploy/presets)。
